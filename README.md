# mnist-ml-blender-gn

`machine_learning_mnist.blend` 本体です<br>

`python\create_digits_dataset_img.py` テクスチャー生成に使用したコードです<br>
`python\simple_nn_ml.py` 今回のGeometryNodesの実装とほぼ同等（のはず）のコードです<br>
